# 🌊 DS-AQUA
A modern, responsive, and visually stunning web application dedicated to showcasing aquatic life and fish species. Built with native web technologies, it features an interactive user experience and clean design.

🔗 Live Demo: [View](https://ds-aqua.vercel.app/)
🔗 Repo Link: [View](https://github.com/dsdipu/ds-aqua.git)

---

## 📌 Overview

DS-AQUA is a frontend-focused web platform designed for aquatic enthusiasts. It serves as an interactive catalog for various fish species, blending smooth transitions with a clean UI. The project focuses on semantic HTML, optimized asset loading, and modular JavaScript to deliver a seamless user experience across all devices.

This project showcases clean directory structures, responsive UI components, and modern styling techniques.

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Variables & Flexbox/Grid)
- JavaScript (ES6+)

---

## ✨ Features

### 🐠 Core Highlights
- **Interactive Marine Gallery:** Structured catalog showcasing different fish types (`images/fishes/`).
- **Brand Integration:** Dedicated space for custom logo assets (`images/logo/`).
- **Responsive Navigation:** Optimized layout for mobile, tablet, and desktop viewports.
- **Dynamic JavaScript Logic:** Handles UI animations, interactive components, or event listening seamlessly via `main.js`.
- **Modular Stylesheets:** Clean and structured CSS approach inside `style.css` for easy scalability.

## ⚙️ Installation & Setup

### Local Setup

1. **Clone the repository**
```bash
    1. git clone https://github.com/dsdipu/ds-aqua.git
    2. cd ds-aqua
```

2. Run Locally
    - Open index.html directly in any browser.
    - Alternatively, use VS Code's Live Server extension for a local development environment.

### Vercel Deployment

1. This project is configured for easy zero-config deployment on Vercel. Simply connect your GitHub repository to Vercel and hit deploy!
